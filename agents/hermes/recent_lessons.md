# Hermes Recent Lessons - 近期教训摘要
更新日期: 2026-05-18

## 当前核心教训

1. **Regime 分类器「盘整盲区」（新/高优先级）**
   - Regime 在低波动/盘整环境中持续错误输出 "good"，导致信号生成器在零交易价值市场中活跃。
   - 这不是经典「趋势中输出neutral」的翻版，而是缺少对「sideways/consolidation」状态的识别。
   - 影响：系统进入空转，所有信号在低活性市场中方向性失效。

2. **幅度低估偏差持续存在**
   - Pred/ATR 全量 < 0.1x，sz300161 低至 0.015x。
   - 模型看到了方向但极低置信度地预测幅度，形成「有方向无力度」的无效信号。

3. **model_score 置信度膨胀（校准问题）**
   - model_score（平均 0.797）与 raw_score（平均 0.472）之间存在约 69% 的差值。
   - 模型对自己的预测质量系统性高估，尤其在低Pred/ATR信号上更为严重。

4. **方向性系统性偏向**
   - 持续性做多偏向：多次审计中信号全部为 LONG。
   - 需要引入对称的空头能力或至少对偏斜程度进行量化监控。

5. **低分+低Pred/ATR双重拦截需求**
   - 当 model_score < 0.70 且 Pred/ATR < 0.02 时应自动零仓位拦截。
   - sz300161（score=0.659, Pred/ATR=0.015x）是教科书式拦截案例。

6. **重复信号衰减机制缺失**
   - 同一标的 2 小时内重复同类信号第二个信号应自动打折。
   - sh603178 在 1 小时内发出 2 个同向 LONG 信号，执行率逐次下降（order_count: 1→0）。

7. **Gate 悖论持续**
   - final_score = raw_score，gate_mult 基本不影响最终评分。
   - 需重构 Gate 逻辑使其真正参与决策而非成为装饰性因子。

8. **信号-执行断裂**
   - 部分信号（order_count=0）永远无法落地为交易，形成预测但未操作的管道问题。

**更新规则**：保持总条数不超过8条，优先保留最严重和最新的系统性问题。
